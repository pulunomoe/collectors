## Installation

### Requirements
- PHP 7.4 (older version might work), with these extensions enabled:
	- curl
	- fileinfo
	- gd2
	- mbstring
	- openssl
	- pdo_sqlite
	- sqlite3
- Composer

### Procedures
1. Run `composer update`
2. Run `run.bat` to start the application
